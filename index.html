快乐与痛苦的相互转化

发表于 2019-12-19 | 分类于 生活

任何一种心情都是可以相互转化的，多大的快乐背面就可能有多大的痛苦，反之亦然。

这事儿就发生在自己身上，新车落地是好事，可没想到，车还没有开出车行，就刮花了，欲哭无泪啊！前一刻的喜悦，马上转变成了巨大的失落感，令我没有想到的是，其实这种失落感存在的期间只有那一刹那，转而我想：反正都是这样了，报保险呗！自己的行为就要承担相应的后果，无论这结果是好是坏，你都无法左右，发生的事情可能扭转，如果从头来一次，谁愿意它发生呢？

所以这个时候，我内心真是想：一切都是最好的安排吧！既然发生了，就由它，算是天意不让我今天来把新车开走，发生这种事，一定有它其他的含义。

我不是唯心主义者，但ACT真的很管用，我今天感觉到了它对我自己的意义所在：发生了事情—接受它—转化它，看看当下真正能做些什么，得到啥教训，而不是怨天尤人。

总结一下：无论发生了什么事情，都要做接受思考，接受它给予我的经验、教训！以后遇到这样的坑就不要重蹈覆辙，下次再开车，我一定会更加小心，无论是在哪里，无论自己一人还是同其他人在一起，决不能分心。这是自己血的教训。过几天再去提车，快乐不就回来了么？以后我不会让任何事让自己深陷而不能自拔，我会全身心投入当下，做好手头该做的事情。说到做到！

幸运的另外一个表述

发表于 2019-12-16 | 分类于 工作

当下每一分的努力，都会在未来你不知道的一个节点用十分的回报给你

2015年，一次偶遇，在集团公司碰见了一个很久没见的老乡，我们一起把酒言欢，畅享当年在一起工作的那些日子，饮酒到凌晨一点，因为在外地，在兄弟面前我老老实实地做了一回客人。后来，我们联手做成了一个大项目，为集团公司和我们分公司带来了不小的收益，中途我们也因为一些不同的意见而不断的交涉，有一两次争得面红耳赤不欢而散。结果是好的，我是下级单位，他是上级单位，我投入了百分之两百的努力，让他也得到了应有的荣誉，再后来，2017年，他升职了，我则回到了原单位，我们单位领导给我庆了功。他升职的第二天，接到他的信息，说感谢兄弟当初的努力，言下之意就是他的升值也有我的一份功劳。我倒觉得没什么，一方面我们是很好的朋友，另一方面，我觉得自己当初就应该那么努力。

前两天，兄弟又电话我，说有个这样的项目，想来想去还是想交给我所在单位来做，但前提是要我亲自跟进，我能感受到他那种非我不可的意思，确实是好项目，我当即答应了下来。接下来，我知道我又要开始新一轮的冲锋，而这付出，我相信肯定至少在明年全年的工作中给我和我的单位带来非常大的好处。

假如没有当初的努力，我的境遇可能不会太差于当下，但还能有拿到这个项目的机会么？答案是否定的！我之前的努力，正在以超越以往的速度和程度回报给我。

我当了“松鼠”很多年

发表于 2019-03-24 | 分类于 学习

我不知道是不是所有松鼠都是这样的，据说松鼠喜欢囤积松果都到了上瘾的境地，而且囤积得这一块那一块的，到最后，冬天过去了，它自己藏的果子有些都找不到了。

我感觉这个很逗乐儿，后来想想，我自己同松鼠很像，自己的NAS和电脑里都囤了不少的“知识”，那些东西是有些是很多年前就存在那里的，当时觉得有用，即使当下没有，也总是觉得以后会有用处。我的微信里和微博的收藏里有很多的文章，有健身的动作，有心灵鸡汤，还有生活的窍门，这些都是我的囤积的“果子”，后来，连自己都忘记掉曾经囤积过它们。真是可笑。

我就在想，囤积这个动作到底有没有用？其实现在如果有东西需要查询，马上百度一下基本可以找到相关的内容，囤积的那些东西的作用，无外乎就是在当时自己觉得“捡到了一块宝贝”，事实上是对自己头脑匮乏的一种明显表现。

因此，我觉得自己有必要重新界定一下，哪些东西是可以进行收藏的，哪些不必。我有必要重新将NAS里3T的内容重新进行个筛选，有用的，留下，无用的，快快扔掉，腾出宝贵的空间，还省多点电钱。

学会生活在高效时间段内——我的番茄工作法

发表于 2018-10-17 | 分类于 学习

如果说之前没有认真思考过如何高效学习，其实是自己没有真实体验和观察自己的所处的生活环境。最近翻出一个曾经淘的番茄时钟，用的次数有限，但电池仍然有电，何不利用一下？我知道番茄工作法的优点，但由于少实践，这两天就发现了不少的问题。

第一个问题就是25分钟之后结束不了。番茄的这25分钟设定是高效的一小段时间，如果停不下来，意味着效率的不断衰减，虽然我自己没有觉察到，而事实上确实如此，既然设置了，必须提醒自己要全力在这25分钟发挥出自己最高的效率，投入全部的精力，如果这样做了，25分钟其实已经可以让自己感觉到稍许的疲倦，尤其是在2个番茄以后。

第二个问题是5分钟回不去。我基本利用这5分钟去上个厕所，泡杯茶，或是看看手机，我发现一拿起手机就麻烦了，有需要回复的信息，有不断冒出的提醒信息，一旦打开微信微博这些东西，基本上就10分钟过去了，再回来的时候需要重新理一下思路才可以。因此我调整了一下，这5分钟我做一下舒展运动，不会离开的太远，最远就到阳台看看远处，尽量放空自己大脑。确保大脑有个有效的恢复，可以使下个番茄开始时，不至于又要重新凝心聚力。

第三个问题是记录的问题。也就是反馈，我发现实物的番茄钟比较直观，但是有关番茄记录的整理就比较麻烦，我是选择按照标准番茄工作法做完一个就画一个叉，四个过后，就忘记了，而且画的地方又不一致，对于自己做过多少个没有个量的统计体系。因此我又到网上搜索了一下，安卓手机下载了疯狂番茄这个APP，里面的体系还算完备，任务分类清晰，做番茄前可以设定任务事项，完成多少个有时间统计，可以做到及时反馈和记录历史的功能。这也就大大方便自己日后的学习总结。

我承认自己有些工具控，所谓工具控就是喜欢瞎折腾，且没有具体目的，纯粹是实现自己凭空的快感，折腾完又不会对工具加以充分的利用。番茄钟也是个工具，让自己可以生活在一个个简短但高效的学习时间内。值得自己慢慢去体会和提高。

我的“罗马”在哪里？

发表于 2018-08-02 | 分类于 生活

有句话这么说：“条条大路通罗马，可是有人就是出生在罗马的”，意思是你穷尽一生也无法达到某些人的起点，或者说你一生为之奋斗的终点对某些人来说其实只是人家的起点。这听起来多少有些悲催。

晚上和老婆去看了电影西虹市首富，整场充满了欢乐的笑声，散场后，我们开车回来路上谈到了两个问题。

特别有钱的人，似乎是干点什么都能赚更多的钱

是啊，你看沈腾那个角色，即使是想败家，都能歪打正着，将看似不可能赚钱的生意做到赚钱的效果，不是他手下多能干，草包只要有梦想，均能化腐朽为神奇。这个道理不理解也要理解，因为现实生活中，大多数人都是朝着赚钱的目标去做，最终达到反效果的太多了。电影中的东西不能太过较真，而生活中，那些手里有钱，股市不好，就投到几套房子里去，过个1年半载，就变成了更多的房子，这其中没有技术含量，明眼人都能看得懂，但能做到的却只有有钱人，这种人，我们碰到过不少。

我开不起酷路泽，但我确信能开得起卡罗拉

回来的路上，前面有一辆非常漂亮的酷路泽，还没上牌，一看就是新车落地。我说，老婆，我估计我这辈子都很难开得上这辆车。我老婆说，你过你的生活，过好你的生活就行了，想这么多有鬼用。没听说么，条条大路通罗马，可是人家出生就在罗马，这能有可比性么？横向对比，你没有其他人富有，这不丢人。纵向和自己比，活得一天比一天好，这才是老正经。

是啊，每个人心中都有个目标，生活得更好，赚更多的钱，拥有更多的资产，这可能是大多数人心中的罗马，我的罗马没有那么多宏伟的建筑，生活充实，过得得劲，就是我的罗马。

我学习的目的是什么？

发表于 2018-07-30 | 分类于 自省

心理学是个好东西，随着时间的推移，越来越发现这个道理是多么的正确。学一样东西就是这样，发现一样东西有用，就好像发现了个宝藏一样，似乎挖下去一定可以寻到宝贝似的。可我为什么要学习心理学呢？我感觉主要有这么几点原因。

一、工作中遇到的问题，确切说复杂的问题越来越多了

大多数人都在不断地发现问题、解决问题的过程中。随着日子的增长，我个人感觉自己要面对和解决的问题的难度也呈现一个上升的趋势。我也曾经认为，工作中没有什么新鲜的问题，大多是前人经历过的，而事实上，很多问题似乎都已经变种，甚至变异，遇到难解的问题请教别人，似乎也是没有遇到过。再用过去的那种方式方法已经行不通，但如何直面问题，直面看待当下的问题，采取何种心态，就成为了一个很关键的解决问题的因素。

二、生活中追求幸福感，需要懂点心理学

对比10年前，其实生活条件已经不错了，可是为什么没有那么强烈的幸福感呢？回到家面对家里的琐事，不想过问，不想说话，不想动呢？曾经恐惧的认为自己得了抑郁症，那段日子睡眠不好，整个人精神状态一塌糊涂，迷迷糊糊的过了几个月，后来，由于工作中的一个突破，让自己又感到生活中没什么大不了的，真为之前那个月的自己可惜，白白浪费了自己的时间。后来我总结了一点，生活中没有什么比时常找到一个小的幸福点更加重要的事情了。

三、学习为的是应用

我曾经删掉了电脑硬盘里的10几个G的学习资料，包括视频教程和电子书，我当时似乎都已经要疯掉，看着那些材料，很明白，自己只是在收集，从来没有认真学过其中的内容，几千本几万本的书籍，一本也没有看完的，删掉后，发现电脑硬盘过一段时间，又积累起一大堆的电子书、教程之类的东西，直到现在，同样是这样的状态。我怀疑自己有收集癖。

偶尔的机会，朋友推荐了《心理控制术》这本书，我就开始了对心理学的兴趣。我就是在不断的重复的犯错循环中，我希望能打破这其中的链条，得以挣脱。后来看到得到APP上上架了一个新课《自我提升心理学》，试听之后感觉挺适合自己，所以又下单交了智商税。我有收集癖，一直到现在，我是个知识的收集人，而不是好的学生。这也是我为什么要继续学习的原因吧，因为我要改变自己，学习的目的就是使用，下载后不会再看的书籍我不会让他在硬盘停留超过一个星期。加油！

坚持的力量

发表于 2018-07-08 | 分类于 生活

说到底，一件事最后成不成事，还是要看我们的坚持程度。小孩子们的学习也大体如此。

孩子准备放暑假了，家长们都在盘算给孩子报什么兴趣班或者学习班，相熟的几个家长都在考虑给孩子报舞蹈、书法班。一些老师时不时给家长们灌输：寒暑假是提升孩子技能的关键时期。是啊，我和我爱人的目标和方向就有2个，一是孩子要得到身体上的锻炼，二是最好学到一些和未来学习相关的技能，比如写字写得更工整一些。

带孩子一边去试课，一边同一同前往的家长闲聊，想当初，我们放假好像真没有什么兴趣班去报的。现在基本上所有的家庭都重视孩子的学习成长，要求也越来越精细化，至少我小时候是没人关注我写字好看与否的。

老板的心情你别猜！

发表于 2018-06-05 | 分类于 工作

与老板的沟通效率和老板当时的心情有非常紧密的联系！
这完全是废话！试想一下自己心情不好，有人找你聊一些事情，你也懒得去打理，这是人之常情。但我们不能天天等着老板心情好才去沟通，毕竟，绝大多数的老板都是结果导向的，有些事情不及时沟通还真没办法达到好的结果。

为什么心情不好会导致沟通效率低？

场景：我不知道老板遇到了什么事情，反正他不爽，我正想沟通一些工作上的问题，突然就想起之前的某件事，借题发挥，给我来了一课。
解析：老板关注他认为非常重要的事！如果我能恰如其分地完成，给出一个比较好的结果，那么，他也就不会去借题发挥了，遇到问题要从自己身上找答案。没错，我们就是那个蛋，无缝隙的话，苍蝇不会盯上你。

总之就是：如果沟通对象心情不在线，有其他事情缠绕，沟通效率肯定不到哪里去，如果平时没有跟进好一些工作，就肯定会被趁机抓住小辫子，这不是老板的问题，任何人都一样。就是这样。
