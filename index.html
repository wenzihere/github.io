一次与五位90后协作的后记（一）

发表于 2018-05-25 | 分类于 工作

单位最近组织了一次活动，上级提出一个课题，要求下面的各级单位组织年轻员工进行一次产品优化展示。这绝对是个展示自我创意和才干的好机会——假如我在年轻个7-8岁！意料之内，我不能上场，但是我接到了一个比上场可能还艰难的任务——组队并带队！

如何选参赛选手

既然是个5人战队，选出的人一定要适合及时沟通，方便聚在一起讨论方案，所以我优先挑选了附近几个小机构里面表现比较突出的年轻人，大致的标准就是要有良好的表达能力、过得去的外表及亲和力、有朝气、有做PPT的才干。

点子从来都是碰撞出来的

在第一次碰面会上，我组织了一次为时1个小时的头脑风暴，我想看看大家对公司的产品的认识程度如何，从我们产品的优点、缺点到客户使用的体验，几轮下来，算是大致了解了这5人对我们产品的认识程度，有些人还是浮于表面，有些人对我们产品的深入了解程度已经表明他们已经做了不少的功课。于是我整理出了几个优化创新方案，其中一个顺应当前智能化的人机交互方案尤为叫好！

将愿景落实才是最大的挑战

没错！回去的路上，我又在思考另外一个问题，我们想到的主意，我们的对手会不会想到？如果大家都想到一起了，我如何用更好的方式来展现出不同进而更加打动评委呢？这个问题让我坚定了两条意见：
1、要采用视频的元素，如果队伍中的那个同事对自己的视频技术稍微有点不自信，我绝对不会采用；
2、一定要有互动。跟台下的互动？如果没有人愿意同我们演讲人员互动，岂不很尴尬？跟自己人互动！因为上台表演的人数是3人，我们可以安排自己人员进行互动。

有必要各个都要“精进PPT”吗？

发表于 2017-12-26 | 分类于 学习

可能是我关注的面儿比较窄，最近看到那么多的贩卖“PPT教程”的，网课一搜一大摞，有大神的、有PPT界的草根明星，当然，他们实力都不凡，看得出来，有些基本就以此为生了，而且，PPT能养活的绝对不止目前这么点儿人。

PPT早就存在了，可很少人去深究一下，这东西是从什么时候开始就突然火起来了呢？究竟是什么原因让这个office软件这么有用处呢？我翻了一些大咖的书籍，基本是速成类的教材，或者是一些绝技，有些还有一些深入的设计理念、美学知识，看来这东西可深可浅。我也喜欢欣赏设计精美的PPT，谈我的一点看法。

PPT曾是某些成功人士的“背景”

或者说得更直接点，有些人是通过使用PPT获得了公众认可并取得了较大的成功的，比如从乔布斯开始的苹果发布会，我在IPHONE4的发布会是真正见识了PPT原来可以这么做，确切地说，发布会开完，我在网上搜索，苹果发布会的背景是什么软件做的？结果告诉我是叫做KEYNOTE的软件，再仔细一下就是苹果版本的PowerPoint。后来我专门找来设计精美的苹果发布会PPT来欣赏，后来，我发现了另外一个神人——雷布斯，没错，要不就不模仿，要模仿就要全套！小米的首次发布会我记得我是在网络上看的，不光是雷布斯的穿着在向乔布斯致敬，他发布会的PPT，也让人感受了我们中华大地上冉冉升起的一颗希望之星！

后来，大家都知道了、习惯了所有发布会，PPT是必不可少的！有些人居然靠PPT造车，你不得不承认，有人拿PPT这玩意儿蒙骗投资人，当然，这不是PPT的错，是人的错。

技多不压人——在看清目标之后

一样东西遭太多人忽视太久之后，在出现了一个巨大的转机之后，大家发现有好处的时候就一窝蜂上，这很正常，但我感觉PPT这东西类似摄影师手中的相机，作为摄影师，器材固然重要，但要表达的东西也就是拍出好的作品仍然最终取决于相机背后的人！所以，PPT很重要，人更是重要，一个好的PPT可以为你的演讲增彩，但它绝对替代不了你真正的演讲实力！所以，在学习前，先弄清楚学来做什么？人家教课可以养家糊口，你要是因崇拜而学习，晚景真的可能不太乐观。

无论学习什么，都是拿来用的，有用处最好，不枉费精力金钱，大学生也真该学点这方面的东西，因为上班后真的可能会遇到做PPT的时候，但这玩意儿并不是一切！你，才是以后的一切！

中年危机？何时不危机？

发表于 2017-12-23 | 分类于 生活

最近被诸如：保温杯、枸杞、油腻等描述中年危机的词给刷屏，本来没啥感觉，这么一整，我估计没危机的人都会感觉到有危机了！

我也想吐槽，人这一辈子，什么时候逃离过“危机”呢？

我曾经的一位领导，感觉压力很大，那年绩效考核上遇到了很大困难，她自己描述曾经同公司领导谈工作谈到崩溃直至大哭，回到家，曾对着上幼儿园的孩子说：妈妈上班好大压力！孩子也被她点到痛处说，我上幼儿园也很大压力啊！那时我刚毕业上班，权当那是个笑话。

上小学三年级的时候，我入选了学校的鼓乐队，打嚓嚓（一种乐器，不知道是不是这么写），就是每天春季运动会，入场仪式上的那种。当时人家都在学习，我却被叫去训练，不但如此，当时和我一起参加训练的我感觉都不是学习怎么好的一群人，而且我在鼓乐队一干就是6年，从小学三年级干到初中毕业，我一直在体验一种“人家上课你训练”的优越感和“被这帮同学给带得不愿意努力学习”的压力感！是啊，真就是这样的，那就是我那个时代的“危机”。

我骨头架子大，力气大，但跑步是问题，初中毕业要体育加试（算入毕业考试总分中的哦，现在的孩子可能没办法想象跑步还要算到毕业分数里去的，说一个更难以想象的，独生子女也可以加分），跑1500米要合格的话得3分47秒（这个数据根据我脑子里最熟的一个数字提取，准确性不保障），那是这个几乎就要我的命了，文化课做的再好，体育加试一项要是落后太多分数，那也完蛋。我清晰记得考试前那2个月我5点起床，脸也不洗就出去到土路上面跑上2公里，回来的时候，鼻孔是黑的，满头满脸的尘土。还好，那时还没有pm2.5这种东西，不然又多了一层压力和危机。

危机从来没有停止过，“生于忧患死于安乐”这句话它对现实的描述甚至高于其哲学和教导意义。我们没有细听过孩子幼儿园的压力在哪里，听过了也权当玩笑，没有想过比我们身体素质差的人是如何鼓起勇气面对未来的考试。说白了，别人的痛不关我事，一群人的痛说出来则可以在心灵上抱团取暖。

后来，我前面提到的那位领导闯过了难关，现在是总公司部门负责人；我也没有因为参加鼓乐队耽误学习，反而交了一大堆朋友；跑步是个难题，努力过了，我没有达到预期目标，但对我考入高中也没有什么实质性的影响。

人生何时逃离过危机？
唯有活在当下，努力奋斗！
一个阶段了一个阶段的活儿！

在王者荣耀中，你得到了什么又失去了什么

发表于 2017-05-06 | 分类于 生活

花了两天的时间，体验了一把当下最火的手游——王者荣耀，这个游戏据说有每天6000万的活跃用户，总用户过2亿，好家伙，游戏的整体体验不用说，精美绝伦，角色设置吸引了各式各样的人群，亲眼看到茶楼、酒吧、咖啡厅、办公室都有人每天捧着手机在忘我奋战。与大学时代联网玩CS不同的是，手游在联网游戏上有了进一步的突破，那就是便利性。CS之类的游戏曾经是成功的，多年不玩，我没有关注它现在如何了，但同传奇一样，王者荣耀在手游领域的成功应该说是更加巨大的，我体验之后就卸载了它，因为我玩的目的仅仅是体验一把，看看究竟是什么让大家如此着迷，顺便总结了一下我的感受。

在现实中突破现实的局限

没错，很多领导也在玩王者荣耀，领导手下的员工也在玩，在现实中你是我的领导，可在游戏世界，你可能只能沦落到被我菜，是不是有一种屌丝逆袭的感觉呢？对！很多企业老板闲暇也玩，还组织员工一起玩，领导冲个几千块钱，在游戏世界里占据绝对优势，领导一帮小弟继续开疆拓土，小弟们俯首称臣，领导您是不是有种众星捧月的感觉呢？

去除游戏系统设置中的一些有失均衡的因素，每个人都在游戏中找到了自己的定位，我是屌丝，我就安心一级一级地打，没有经验翻倍，没有点卡充值，但我也可以通过通关和任务拿到不少的金钱，有钱人有有钱人的玩法，追求的东西不同，咱就不会去计较，也不应该去计较，否则就没办法玩了。这个游戏照顾的真是全面：来吧，有钱你就充值吧，没钱也没事，我安排些和你一样的人一起玩，层次不同，我给你应该有的体验。

游戏真正变成了一场社交

单机版和pc版游戏让人在社交上不方便是不言而喻的，要想和同伴玩，你得到网吧联网，现在告诉你，手机也可以，而且不用怎么费流量就可以有一个不错的体验。游戏成了一种非常的社交方式和手段。同事朋友在席间饭后，在厕所、公交站，随时都可以撸上一把。方便，微信和微博是图文记录朋友陌生人现在或者之前做过什么，而手游，则把你和对方连接到一起，每一场游戏里，都记录了你和对方共同经历了什么。还有什么比这个更能称得上是社交的呢？

得到之后

好了，我现在可以说，在得到了现实中不能体会的一些体验，在和好友共同亲历了开疆拓土之后，我累了，一晃就凌晨一点了，一下子一个下午就过去了，手机电池插了几次电源，手机都可以用来煎鸡蛋了，到了该洗洗睡的时候，才发现自己已经过了几个小时，就像经常抽烟的人回首过往才发现，如果不吸烟，那些烟钱都可以买辆车的感觉一样。

这个世界就是这样，得到就会失去，如果游戏可以自制，就没有那么多沉迷其中的人，这是值得警醒的。

所以，我体验到了，就卸载了王者荣耀，因为这个游戏太成功了，我真的无法保证自己每天能够在该休息的时候及时放下手机。再来一局的选项太具有吸引力。

绝不为他人背书！

发表于 2017-05-04 | 分类于 工作

做信贷，发生风险事件，是好事还是坏事？站在自身成长的角度，当然不是什么好事，但从另外一个角度来看，未必不是一件好事。从中反省，就可以发现这么一个深刻的问题，那就是在生活中，无论如何都不能为他人背书！无论是出于什么原因、对方是谁！

事实

A厂老板算是H的老友，算起来怎么都有20多年的交情了，A老板的性情爱好，H心中有数，出于争揽业务考虑，H向行里推荐了A厂做贷款2000万，后来A厂老板资金链断裂，无法偿还银行贷款，最终形成了不良。

总结

一、无论是谁推荐的业务，银行就是应该要按照既有的信贷流程核定贷款，过去粗放型经营的年代，靠一张上级领导的条子就可以放款的年代是一去不复返了。做业务有章可循，有法可依，这为银行从业人员画了一道防火墙，它也是一条信贷从业人员的底线。
做这笔业务的过程中，信贷员小L按照银行贷前调查的流程，充分论证了可贷款额度，并在授信审批之后按要求落实了前提条件，争取到了土地厂房作为抵押并办理了正常的抵押登记手续。贷后管理中有发现其在破产之前经营出现异常的情况记录，并及时报送了重大事项，为行里改变对该司的授信政策提供了重要和及时的参考依据。小L并没有顾忌到A厂是H的老朋友就突破底线，做一些违反银行信贷原则的事情，非常好，这也就为他日后的责任认定打下了一个免责的基础。过程做的正确，尽到应尽的义务和责任，可以免责。小L做得好！

二、从生活常识的角度看：绝不为任何人背书！记得之前H在推荐之前说，这个老朋友的信用他可以百分百的肯定不会欠银行一分钱！可结果呢，在血淋淋的事实面前，他也不得不承认：当初的想法是错误的。其实错的不是他，是他那脑子里的固有的想法：以为对对方了解的够深，甚至可以拿人格和个人信用为其担保，实则不然，牢记吧：不要为任何人任何事做个人背书！因为没有人能够可以为其他人承担责任。

个人信用越来越重要的时代已经来临，如果没有意识到这一点，企业还是以以前那种粗放型的经营方式做，要么这个企业长不大，要么半路夭折，最惨的是永无东山再起的机会，这是彻底的被PK出局的结局！

健康是享受生活的必要条件

发表于 2017-03-01 | 分类于 自省

今天到一个企业拜访，席间说起养生，该企业财务总监现在瘦得很，而在5年之前他足足比现在重30斤，鬼知道他经历了什么。

拉家常谈到养生，该总感叹道：只有经历过病痛，才能感受到，什么房子、车子、钞票，全是扯淡。我喝酒把甲状腺喝挂掉了，过了两年我又把胃喝到让我痛不欲生，现在我是荤的不能吃、辣的不能吃、过咸的不能吃，出去吃饭，满桌子的菜我能下筷子的就马上筛出去了2/3，你说痛苦不？这胃病治了3年了，才刚恢复了5成左右，现在我别无他求，能让我安安稳稳痛痛快快地吃上饭就行了。

屋外的雾霾我们没办法彻底除掉，屋内的环境要注意

该总家里有两位医生，一个是他老爸，一个是他大哥。他老爸死于肺癌，原因是二手烟和炉灶粉尘，长久的吸入导致了最终的悲剧。过年回家，他大哥拉着他给他上课：必须要注意避免二手烟！避免粉尘，外面大气污染我们没有办法，但室内空气质量却可以控制，保持流通，购置个空气净化器，善待自己的肺。

压力谁都有，但不一定人人都能处理好

该总二哥是银行的一位领导，近年来经济环境不好，导致了银行出现了不少的烂账，出现一笔罚一两个月的工资，要是多几笔，那一年下来基本白干了。他也坦然，刚开始自己确实想不开，整晚睡不着觉，思前想去如何面对这些问题，可最终苦了害了自己。自己想想不是么？银行贷款出去就得承担无法收回的风险，出现了风险，只要自己所有的业务流程合法合规，客户还不起钱就照走法律流程嘛，想那么于事无补。确实现在一些人面对这些问题可能真的一时无法接受而自己瞎琢磨，最终因心理压力过大健康就亮起了红灯。想想吧，本来业务已经赔了钱了，我们不能再把自己赔进去，一两个月的工资算什么？扣一年的工资能让你家庭破裂么？能多大程度上影响你的生活质量？任务指标完不能又能怎么样？你是没有尽力么？如果尽力了，结果不能接受又能怎么样？

二月的事儿忒多，春节大吃大喝，回来加班加点。熬夜，缺乏运动……这些都是给我们的健康做减法。唯一正确的道路是：远离烟酒、熬夜，多运动，保持旺盛的生命力来享受人生。我们这一代人，活到90岁算是正常，60、70岁就挂了的话，人家会笑话的。

过不过节在“心”，不在“假期”

发表于 2017-02-09 | 分类于 生活

如期而至的春节，一成不变的春运，随着烟花爆竹的烟雾散去，春节也就这么过去了，年初的主题离不开春节假期，就像假期的主题离不开吃、喝、睡、玩一样。当假期余额不足之时也大抵是口袋里的钱需要充值的时候。我特别理解《逃回北上广》那篇文章里说的：带着回来时一样的坚毅的眼神，望着返程的路。不回真不行，一来家这几天给予我们的已经似乎足够了，二是我们实在也很难适应乡村里的生活了，说来脸红，我居然内心感觉到外面的优势似乎好过家里，对此甚至感到深深的羞愧，虽然不知道愧对谁了。

父母白发又多了几条

明显感觉到55岁以后的时间里，父母的白发一年比一年多。可能受累于长年累月的农村生活，年轻时透支的精气神在上了年纪就全部暴露出来了，老妈妈闲下来后一手揽着孙女儿，一边跟我说：“谁不都得要老嘛，都说我们老得快，你看我孙女儿都这么大了，时间过得快呀！”要不是老妈妈和老爸不习惯这城市里的生活，我早就把他们从农村接出来了。我说接她出来，她又总是说：“我哪里能走出去？哪里又能比家里好哇，况且，你姥姥还在，我哪里也不去……”也对啊，我妈是幸运的，有了孙女儿，自己身体健康，她还有自己的妈妈在身边可以照顾。

老爸喜欢喝酒，我就陪他喝酒，好酒一般他都留着等我回家一起跟他喝，可上桌吃饭的时候，总是自己喝他那从集市上打来的散酒，还说好喝。我说：爸，过年了，咱们都喝好点的酒吧。他却说：我习惯这一口儿，你喝你的。这几年我不敢往家里带酒了，因为有一年过年回家，我带了两瓶五粮液回来给他喝，过了几个月，我妈电话我说我爸他戒酒了，当时第一个念头冒出来就是：我是不应该给爸喝那五粮液的，喝着时候确实很好，但人活着什么都有个“量”，一个人喝好酒也应该有个量，量达到了，基本也就是告别的时候了，当然这是我当初迷信的想法，后来老爸就自然而然的好了，我这颗心才算放下。我爸是个土生土长的农民，好酒要留着给他慢慢喝，做儿子的就希望他能够长命百岁，哪怕喝散酒，长饮常有岂不乐哉？

村子仍然还是那个村子

变化的是车、车、车！
农村有车也就是这三两年的事儿，从之前的屈指可数的几辆到现在，很多人家的门口都停着车。不经常回来的我们更是感受到了现在农村的变化，特别是人们生活条件的变化。真是很多人都有钱了。只要你不闲着，肯劳动，就肯定饿不死而且生活得很好。我妈是这么说道的，这道理拉到城市里其实也一样。

农村的孩子

姐姐妹妹家都有两个孩子，二胎政策在农村真正是落地开花结果了。不像城市，很多人顾忌生活质量下降，顾忌孩子的教育培养成本高。我姐私下里问我什么时候要二胎，我笑着说等再赚多点钱，钱是啥，有用处就自然有来源，这么年轻，还愁赚不到钱？我姐就是这么个思路。她还给我算计一个农村孩子的花销真的不比城市里的低，样样东西都是买最好的，言外之意就是说：我们农村的可以响应号召生二胎，你城市的怕什么！对，我同意我姐的说法，可我就是想再多赚点钱，增强点安全感。

假期结束，春天来临，万物又开始进入一个新的循环周期。快乐！

疑人不用，用人不疑

发表于 2017-01-21

明天启程回家过年，工作还没有结束，中国人的传统是这样的，2017年设一个小目标：明年争取让远在北方的爸妈能够在自己身边一起过年。这几天就暂停更新博客，内容只能靠手机来写，回来后再更新。预祝所有人新年快乐！>

“爸爸是谁给你打电话啊？是不是李老板呐？”我闺女昨晚在我打完电话后这么问我，我先是一怔，然后和老婆一起哈哈大笑。

“对对对，可不是李老板嘛，整天给我打电话，让我干活儿！”我笑哈哈的回答闺女。

很多事情，大人可能没想透，可孩子能察言观色，看得真叫一个准！光头强的李老板每天电话督促他砍树，我的电话那端的“李老板”也一样，天天督促我的业务指标，和光头强比起来，我好想我也好不到哪里去。

工作必须要投入，这是我自己对工作的一个信念和原则。上级领导的督导我也能够认真去听取并执行，因为机构的方针政策是大指针，只有同心同德才可能达成最终绩效目标。我想说的是在这个过程中这种对下级督导的“力度”问题。

不要寒了将士们的心

对下级督导的力度是个技术活儿，太大力或不对路数，容易让下级觉得“上面人不信任我”，既然他能够在目前的岗位任职，我们首先要确认他是有这个能力做好岗位工作的，上级需要做的不是天天耳提面命，电话信息漫天飞舞，一方面这渲染了工作超高的工作压力，另一方面容易真的让人觉得领导不信任我。想想看，你的下级是没有尽力干活儿么？是在藏着掖着么？如果不是在关键时点上，下级可能不出力干活儿么？不会的。紧张只能在关键时刻用，所谓好钢用在刀刃上，你天天将弓箭拉到尽，那么该使出力气最后可能也不能了。而且长期这么做更严重的后果可能寒了将士们的心。

疑人不用，用人不疑

你不是怀疑人家的能力和态度，那么你就应该全力支持他去做，领导看的是结果，都明白，但过程中作为领导，你提供了什么支持？还是一味的拿着鞭子吆喝？你下属是一群活生生的人还是一帮驴马供你驱使呢？

以后的工作中，自己也要警惕，我要督促完成的目标暂时遇到困难或者无法达成的时候，首先要看作为上级，是哪里没有部署好，还是提供了太少的帮助给下面人。避免那种拿着鞭子吆喝的情况发生。

领导有多面性，他们的要求无穷无尽，我们做出来的永远赶不上领导的要求，在做好岗位职责的同时，尽善尽美是我们工作中的另一个重要原则。但愿所有人都能在压力之下保存一个完整的自我。如果不是那样，我想迟早会变成一个麻木不仁的人。
