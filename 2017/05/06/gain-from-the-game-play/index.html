在王者荣耀中，你得到了什么又失去了什么

发表于 2017-05-06 | 分类于 生活

花了两天的时间，体验了一把当下最火的手游——王者荣耀，这个游戏据说有每天6000万的活跃用户，总用户过2亿，好家伙，游戏的整体体验不用说，精美绝伦，角色设置吸引了各式各样的人群，亲眼看到茶楼、酒吧、咖啡厅、办公室都有人每天捧着手机在忘我奋战。与大学时代联网玩CS不同的是，手游在联网游戏上有了进一步的突破，那就是便利性。CS之类的游戏曾经是成功的，多年不玩，我没有关注它现在如何了，但同传奇一样，王者荣耀在手游领域的成功应该说是更加巨大的，我体验之后就卸载了它，因为我玩的目的仅仅是体验一把，看看究竟是什么让大家如此着迷，顺便总结了一下我的感受。

在现实中突破现实的局限

没错，很多领导也在玩王者荣耀，领导手下的员工也在玩，在现实中你是我的领导，可在游戏世界，你可能只能沦落到被我菜，是不是有一种屌丝逆袭的感觉呢？对！很多企业老板闲暇也玩，还组织员工一起玩，领导冲个几千块钱，在游戏世界里占据绝对优势，领导一帮小弟继续开疆拓土，小弟们俯首称臣，领导您是不是有种众星捧月的感觉呢？

去除游戏系统设置中的一些有失均衡的因素，每个人都在游戏中找到了自己的定位，我是屌丝，我就安心一级一级地打，没有经验翻倍，没有点卡充值，但我也可以通过通关和任务拿到不少的金钱，有钱人有有钱人的玩法，追求的东西不同，咱就不会去计较，也不应该去计较，否则就没办法玩了。这个游戏照顾的真是全面：来吧，有钱你就充值吧，没钱也没事，我安排些和你一样的人一起玩，层次不同，我给你应该有的体验。

游戏真正变成了一场社交

单机版和pc版游戏让人在社交上不方便是不言而喻的，要想和同伴玩，你得到网吧联网，现在告诉你，手机也可以，而且不用怎么费流量就可以有一个不错的体验。游戏成了一种非常的社交方式和手段。同事朋友在席间饭后，在厕所、公交站，随时都可以撸上一把。方便，微信和微博是图文记录朋友陌生人现在或者之前做过什么，而手游，则把你和对方连接到一起，每一场游戏里，都记录了你和对方共同经历了什么。还有什么比这个更能称得上是社交的呢？

得到之后

好了，我现在可以说，在得到了现实中不能体会的一些体验，在和好友共同亲历了开疆拓土之后，我累了，一晃就凌晨一点了，一下子一个下午就过去了，手机电池插了几次电源，手机都可以用来煎鸡蛋了，到了该洗洗睡的时候，才发现自己已经过了几个小时，就像经常抽烟的人回首过往才发现，如果不吸烟，那些烟钱都可以买辆车的感觉一样。

这个世界就是这样，得到就会失去，如果游戏可以自制，就没有那么多沉迷其中的人，这是值得警醒的。

所以，我体验到了，就卸载了王者荣耀，因为这个游戏太成功了，我真的无法保证自己每天能够在该休息的时候及时放下手机。再来一局的选项太具有吸引力。
