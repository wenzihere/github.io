过不过节在“心”，不在“假期”

发表于 2017-02-09 | 分类于 生活

如期而至的春节，一成不变的春运，随着烟花爆竹的烟雾散去，春节也就这么过去了，年初的主题离不开春节假期，就像假期的主题离不开吃、喝、睡、玩一样。当假期余额不足之时也大抵是口袋里的钱需要充值的时候。我特别理解《逃回北上广》那篇文章里说的：带着回来时一样的坚毅的眼神，望着返程的路。不回真不行，一来家这几天给予我们的已经似乎足够了，二是我们实在也很难适应乡村里的生活了，说来脸红，我居然内心感觉到外面的优势似乎好过家里，对此甚至感到深深的羞愧，虽然不知道愧对谁了。

父母白发又多了几条

明显感觉到55岁以后的时间里，父母的白发一年比一年多。可能受累于长年累月的农村生活，年轻时透支的精气神在上了年纪就全部暴露出来了，老妈妈闲下来后一手揽着孙女儿，一边跟我说：“谁不都得要老嘛，都说我们老得快，你看我孙女儿都这么大了，时间过得快呀！”要不是老妈妈和老爸不习惯这城市里的生活，我早就把他们从农村接出来了。我说接她出来，她又总是说：“我哪里能走出去？哪里又能比家里好哇，况且，你姥姥还在，我哪里也不去……”也对啊，我妈是幸运的，有了孙女儿，自己身体健康，她还有自己的妈妈在身边可以照顾。

老爸喜欢喝酒，我就陪他喝酒，好酒一般他都留着等我回家一起跟他喝，可上桌吃饭的时候，总是自己喝他那从集市上打来的散酒，还说好喝。我说：爸，过年了，咱们都喝好点的酒吧。他却说：我习惯这一口儿，你喝你的。这几年我不敢往家里带酒了，因为有一年过年回家，我带了两瓶五粮液回来给他喝，过了几个月，我妈电话我说我爸他戒酒了，当时第一个念头冒出来就是：我是不应该给爸喝那五粮液的，喝着时候确实很好，但人活着什么都有个“量”，一个人喝好酒也应该有个量，量达到了，基本也就是告别的时候了，当然这是我当初迷信的想法，后来老爸就自然而然的好了，我这颗心才算放下。我爸是个土生土长的农民，好酒要留着给他慢慢喝，做儿子的就希望他能够长命百岁，哪怕喝散酒，长饮常有岂不乐哉？

村子仍然还是那个村子

变化的是车、车、车！
农村有车也就是这三两年的事儿，从之前的屈指可数的几辆到现在，很多人家的门口都停着车。不经常回来的我们更是感受到了现在农村的变化，特别是人们生活条件的变化。真是很多人都有钱了。只要你不闲着，肯劳动，就肯定饿不死而且生活得很好。我妈是这么说道的，这道理拉到城市里其实也一样。

农村的孩子

姐姐妹妹家都有两个孩子，二胎政策在农村真正是落地开花结果了。不像城市，很多人顾忌生活质量下降，顾忌孩子的教育培养成本高。我姐私下里问我什么时候要二胎，我笑着说等再赚多点钱，钱是啥，有用处就自然有来源，这么年轻，还愁赚不到钱？我姐就是这么个思路。她还给我算计一个农村孩子的花销真的不比城市里的低，样样东西都是买最好的，言外之意就是说：我们农村的可以响应号召生二胎，你城市的怕什么！对，我同意我姐的说法，可我就是想再多赚点钱，增强点安全感。

假期结束，春天来临，万物又开始进入一个新的循环周期。快乐！
