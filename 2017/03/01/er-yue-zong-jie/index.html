健康是享受生活的必要条件

发表于 2017-03-01 | 分类于 自省

今天到一个企业拜访，席间说起养生，该企业财务总监现在瘦得很，而在5年之前他足足比现在重30斤，鬼知道他经历了什么。

拉家常谈到养生，该总感叹道：只有经历过病痛，才能感受到，什么房子、车子、钞票，全是扯淡。我喝酒把甲状腺喝挂掉了，过了两年我又把胃喝到让我痛不欲生，现在我是荤的不能吃、辣的不能吃、过咸的不能吃，出去吃饭，满桌子的菜我能下筷子的就马上筛出去了2/3，你说痛苦不？这胃病治了3年了，才刚恢复了5成左右，现在我别无他求，能让我安安稳稳痛痛快快地吃上饭就行了。

屋外的雾霾我们没办法彻底除掉，屋内的环境要注意

该总家里有两位医生，一个是他老爸，一个是他大哥。他老爸死于肺癌，原因是二手烟和炉灶粉尘，长久的吸入导致了最终的悲剧。过年回家，他大哥拉着他给他上课：必须要注意避免二手烟！避免粉尘，外面大气污染我们没有办法，但室内空气质量却可以控制，保持流通，购置个空气净化器，善待自己的肺。

压力谁都有，但不一定人人都能处理好

该总二哥是银行的一位领导，近年来经济环境不好，导致了银行出现了不少的烂账，出现一笔罚一两个月的工资，要是多几笔，那一年下来基本白干了。他也坦然，刚开始自己确实想不开，整晚睡不着觉，思前想去如何面对这些问题，可最终苦了害了自己。自己想想不是么？银行贷款出去就得承担无法收回的风险，出现了风险，只要自己所有的业务流程合法合规，客户还不起钱就照走法律流程嘛，想那么于事无补。确实现在一些人面对这些问题可能真的一时无法接受而自己瞎琢磨，最终因心理压力过大健康就亮起了红灯。想想吧，本来业务已经赔了钱了，我们不能再把自己赔进去，一两个月的工资算什么？扣一年的工资能让你家庭破裂么？能多大程度上影响你的生活质量？任务指标完不能又能怎么样？你是没有尽力么？如果尽力了，结果不能接受又能怎么样？

二月的事儿忒多，春节大吃大喝，回来加班加点。熬夜，缺乏运动……这些都是给我们的健康做减法。唯一正确的道路是：远离烟酒、熬夜，多运动，保持旺盛的生命力来享受人生。我们这一代人，活到90岁算是正常，60、70岁就挂了的话，人家会笑话的。
