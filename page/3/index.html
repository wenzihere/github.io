翻篇2016，迎接2017！

发表于 2016-12-31 | 分类于 自省

年底的总结每年这个时候还是要做的，对比去年此时，该实现的基本已经达成，没有实现的，现在着手也来不及了，时间就这样，不管你愿不愿意，它都会慢慢地离你而去，毫不留情。就像那句话说的：来不及细细端详，就要挥手告别。

2016年，悲喜交加，喜乐参半，无论工作和生活，都是这样，延伸开来，也许这就是生活：你可以计划未来，但无论是谁，都很难预测到结果，近的一周都难预测，更何况在年头的时候预测年底？现在的结果和当初的计划间的距离就是对自己的期盼和自己本身的实际能力间的距离，我们都要努力提高自己，所以，一句话来总结：我还有很大空间（希望明年的时候真的不要再用这样的话。）

工作上
作为一个职业人的感觉应该是已经找到了，虽然这耗费了我不少的时间和精力，从控制自己的情绪到控制自己的语言和行为，我想自己有了很明显的进步。有时候沾沾自喜，其实这就是根源。平心静气的就一件事来同任何人交换自己的意见，让自己的不遮遮掩掩地做自己喜欢做的事情，给予身边人力所能及和必须的帮助，帮助他人成长，我想我做到了。

需要改进的地方：
①团队的每个人并没有完全均等地给予自己的指导。犯懒是一个原因，忙也是一个原因。但无论如何都应该至少同团队的每个人在每个月都有单独的交流和沟通。我遇到的很多难题其实都可以通过这个环节提前预知和解决掉的；
②解决问题的能力还要抓紧时间提升，并不是说自己能够解决所有问题，但能够做到的和应该做到究竟为什么最后却没有达成？归根到底还是在行进过程中，自己解决问题的能力有限。
③盯紧项目。千万不能拖延，有时就那么一下子，机会过去了就过去了，等再回神的时候，一切都已经晚了。

生活上
孩子教育：要学习！学习！再学习！遇到的问题都是普遍的，能够在孩子面前控制好自己的情绪，当自己生气的时候，告诉自己：她会长大的！你也有过童年！
健康维护：注意休息！要注意职业病，继续坚持跑步，尝到了跑步的益处，必须要坚持下去！尽量少吃高蛋白食物。
投资管理：看好自己的口袋。稍一放松，净值下降就容易被忽略，吃的亏太多了，要吸收教训了！
学习：时刻都在进行！不要只关注自己读了多少本书，用才有用，多问自己：这本书能够给我带来什么改变？我是否应该投入时间在手里的这本书上？

其实每一天都是新的起点，如果有决心和信心改变，就应该时常自省，无论面对什么困难都能够想到应对的预案的！加油2017！

拖延症的成因——“懒”

发表于 2016-12-29 | 分类于 自省

治好任何疾病都需要知道这种疾病的成因是什么，拖延症只是个比较广泛存在的症候群之一，而危害却非常大，如果不及时将自己的拖延习惯改一改，真的可能会影响到生活的方方面面。

“懒”的成因
有一句话特别好：什么岁月静好，你特么就是懒！
这句话基本上说出了拖延症的表面成因，懒！不懒就不会拖延，不会形成拖延的习惯，不会得了拖延症。这成因特别直接，直接到有时候自己对自己都无法忍受的时候就大喊一声：“我特么怎么就这么懒！”然后发毒誓，我一定要变得更加勤劳一些，可是我们要知道：由勤入懒易，由懒入勤难啊！这是我最深刻的体会，有这么几个例子可以印证一下。

少去做"无用功"

发表于 2016-12-25 | 分类于 自省

一不留神，一个小时候过去了，一个上午过去了，躺在床上，一天过去了。

李笑来老师在《把时间当做朋友》这本书给我最大的一个启发就是凡是要顺其自然，向内关注而不要刻意去进行所谓的时间管理，因为时间是无法管理的，无论如何，它都要流走。

临近年底，感觉时间不够用的焦虑与日俱增，每天好像一睁眼，就要开动大脑去处理一大堆的事情，时间表基本爆表，当然，全身心投入工作的一个比较明显的地方就是：时间过得特别快

低效率的呆在电子产品前，还不如让自己的大脑休息一会

我发现，自己的效率之所以低，很多时候与手机／电脑的信息提示有很大关系，一会看看朋友圈，一会要恢复回复某个人某个群的信息，尤其是群信息，耗费了我大量的注意力去应对，还有微博。其实大部分的信息都可以在做完手头工作后再去回复，微博和微信朋友圈只能在休息的时间去瞄一眼的。

懂得对周围对人say no！

没办法，酒局饭局肯定会有，但是不是一定要牺牲一个晚上和女儿玩的时间来陪对方呢？
答案是大可不必！只有在恰当的时间做恰当的事，才能工作生活平衡。才能让自己更加游刃有余，如果有一个比较合理的理由，我相信大家都会理解。

低头做事，更要“抬头看路”

发表于 2016-12-15 | 分类于 工作

工作和生活中遇到的事情都没那么简单，但也绝不会复杂到哪里去（我是说大多数人遇到的大多数情况）。

感觉很多时候之所以认为事情复杂，是因为我们太关注低头做事，没顾得上抬头看路。今天就碰到了小伙伴“掉坑里”的例子。

A姑娘，我同事，四川妹子，入职5年，性情爽朗，做事干练，但就是有时候容易急躁（通常这种说法用于在自我鉴定中），早晨接收到一份任务，她采用的工作流程是这样的：一看，做过，好！回到工位马上翻出之前的方案文档，对着电脑就是一个上午，照猫画虎得做下来，下午上班时候将成果交给了领导，领导一看她拿着方案来了，首先就觉得：嗯，效率蛮高嘛，再打开文稿一看，咦，怎么用这个模板做的，我们上个月不是刚刚更新来这种任务的模板文件了么？大的框架结构用错了，只好回去重头按照新模板来做。

我同事小A姑娘当时的委屈和无奈想象得出，浪费了一个上午的时间，能怪谁？我只能安慰似的劝说她几句罢了。

回顾一下：小A在接受任务时，根本就没去从头梳理任务流程，而是直接走捷径：寻找以前做过的案例，照猫画虎的操作，连基本的公司固定格式的范本文件有没有更改都没有去查询，或者跟周围的同事问一下，时间浪费在一个上午不说，还要重头来过。

这样令人捶胸顿足的事情所有人都有可能犯过，我也犯过，总结一下就是只顾低头做事不抬头看路的结果大体一样：轻者会像小A一样做重复功、无用功，重者会费力不讨好，理解你的领导和同事懂你犯了个不是很严重的错误，不理解你的领导劈头盖脸地骂你一顿，那就太不值得了。

我是怎么沾染上烟瘾的？

发表于 2016-12-14 | 分类于 自省

我有13年的烟龄……对此，我真的感到很难接受。我深信改变是这个世界永恒的主题，唯有自省方能自知，进而改变。

一、接触到香烟的初级阶段

2004年1月，寒假结束，我从家乘坐绿皮车回武汉继续大四最后半个学期的课程，颠簸了8个多小时后来到了中途邯郸姑姑家。跟姑姑谈到未来毕业找工作末了，姑姑说：“带上两条烟回学校吧，万一找工作的时候遇到要抽烟的领导呢？可能用得着，如果用不着就带回家给你爸抽。”我也没说什么，我得承认，在此之前，由于上学期间我身体不是特别好，经常感冒咳嗽，我坚定的认为我这一生不可能与香烟有交集，我父亲有重度烟瘾，我妈也曾告诫我说，你将来决不能跟你爸一样学抽烟。想想那时，我那种对香烟不屑的表情和“我绝不可能抽烟”的豪言壮语，啧啧……哪能想到今天的自己呢？所以千万别说大话！无论在哪个年龄段，面对什么样的人，至今我对我母亲在吸烟这件事上对曾经的“豪言壮语”仍旧愧疚不堪。接着，这两条烟就放进了我的行李箱，跟我一起来到了学校。现在回想起来，就是这两条烟铺就了我通向吸烟这条路的轨道，就是这两条烟，拉我进入了长达13年的吸烟历程。﻿
